Protótipo: Portfólio de Equipe (Hugo)
Este é um protótipo de um site de portfólio de equipe, desenvolvido como parte de um projeto de Gestão de Projetos de Software.

O site apresenta um guia sobre a ferramenta Hugo (na página principal), uma página "Sobre o Projeto" e os perfis dos integrantes da equipe (Lizandra, Bruna e Gabriel).

 Site Ao Vivo (Deploy)
O deploy (publicação) deste protótipo está a ser feito automaticamente pelo Netlify e pode ser acedido aqui:

https://hivegps.netlify.app/

 Tecnologias Usadas
Gerador de Site: Hugo (SSG)

Tema: hugo-brewm (instalado como um Git Submodule)

Deploy (CI/CD): Netlify (configurado via netlify.toml)

Controlo de Versões: Git & GitHub
